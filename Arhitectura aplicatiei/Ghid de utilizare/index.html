Descriere

Cric Alert este o aplicatie web care ofera mai multe functionalitati cum ar fi trimiterea de alerte, cautarea persoanelor disparute, vizualizarea evenimentelor de pe harta in timp real, adaugarea de evenimtente pe harta.

Cum instalam CriC?

Din fericire CriC este o aplicatie usor de instalat. Tot ce trebuie sa faceti este sa adaugati folderul cu proiectul in interiorul unui folder git si sa adaugati tabelele cu utilizatori, cu numere de telefon si cu adrese de email.

Pagina principala

Pagina principala contine o scurta descriere si un formular pentru autentificare. Formularul este vizibin numai atunci cand utilizatorul este delogat. In momentul in care acesta se autentifica, formularul dispare.

Pagina Harta

Afiseaza o harta cu evenimentele.

Pagina Persoane Disparute

Ofera posibilitatea utilizatorului sa ofere informatii despre o persoana pierduta sau sa caute pe cineva. Pagina este conectata la serverul google, deci informatiile pot fi incluse si de pe alte pagini. Pentru a cauta o persoana in baza de date, trebuie introduse numele si prenumele si apasat pe butonul submit. Aplicatia va afisa informatii despre persoana cautata. Pentru a adauga informatii despre o persoana trebuie sa completam formularul din sectiunea ofera informatii.

Pagina Localizare Evenimente

Ofera posibilitatea de a introduce, de a vizualiza si de a sterge evenimente introduse.

  • Pentru a introduce elemente selectati una distre optiunile gasite in partea dreapta a paginii(Cutremure, Inundatii, Alunecari de tere, Incendii), completati campurile si apasati submit. Datele vor fi trimise catre serverul google si vor fi afisate in partea din stanga a paginii.
  • Pentru a sterge un element apasati pe iconita din dreapta lui.

Pagina Profil

Afiseaza informatii despre utilizator. Pentru a modifica informatiile completati campurile si apasati submit.

In cazul in care utilizatorul este administrator, acesta poate trimite alerte sau sterge utilizatori apassand pe cele doua butoane corespunzatoare actiunii pe care doreste sa o execute.

  • Dupa ce este apasat butonul Trimite Alerta se deschide o pagina noua unde se poate adauga un mesaj care va fi trimis prin sms si prin email atunci cand butonul trimite este apasat.
  • Pentru a sterge un utilizator, administratul trebuie sa apese pe iconita corespunzatoare fiecarui user.

Pagina Inregistrare

Este vizibila numai atunci cand utilizatorul este delogat.

Permite inregistrarea utilizatorilor cu dreptul de adminstrator. Acestia vor avea dreptul sa adauge evenimente, dar nu vor putea trimite alerte.

Cum adaugam un adminstrator nou?

Din pagina phpmyadmin selectati tabelul Useri, iar in coloana tip modificati valoarea din user in admin pentru fiecare utilizator care va avea dreptul de admin.
